# Иллюстрация: Ноты для пианино

![Ноты для пианино]("Ноты для пианино.png")

Обозначения:
* ск - субконтроктава,
* к - контроктава,
* Б - большая октава,
* м - малая октава,
* 1 - первая октава,
* 2 - вторая октава,
* 3 - третья октава,
* 4 - четвертая октава,
* 5 - пятая октава.

Иллюстрация выполнена в LibreOffice Draw 5.1.4.
